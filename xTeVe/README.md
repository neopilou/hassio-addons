# Hass.io Add-on: xTeVe
xTeVe Server

### About
This add-on add xTeVe server. It does not support SSL.

### Installation
1. Add the add-ons repository to your Hass.io instance: `https://github.com/neopilou/hassio-addons`
2. Install the xTeVe add-on
3. Start the add-on
