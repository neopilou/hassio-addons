# Hass.io Add-on: Tvheadend
Tvheadend Server

### About
This add-on add Tvheadend server

### Installation
1. Add the add-ons repository to your Hass.io instance: `https://github.com/neopilou/hassio-addons`
2. Install the Tvheadend add-on
3. Start the add-on 

### Configuration

Nothing :)

### Access

- HTTP : hass-url:9981
- HTSP : hass-url:9982
